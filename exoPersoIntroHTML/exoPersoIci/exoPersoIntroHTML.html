Comment insérer des images ?

les différents types d'image qui sont le plus souvent utilisés en HTML sont :

  1. Le type "jpg" qui est un format conçu pour réduire le poid des fichiers photos.
    La compression du "jpg est généralement plus élevée que celle du "png".
    Ainsi, une photo enregistrée en "jpg" s'affichera plus rapidement que le format "png".
    Il est donc plus conseillé d'enregistrer les photos en "jpg.

  2. Le format "png" qui est adapté à la plupart des graphiques, logos ou dessins.
    Ce format est généralement utilisé pour tout ce qui n'est pas une photos
    et il a l'avantage de garder la transparence de certaines images..

  3. Le format gif est assez vieux, mais pratique quand on veut pnctuer notre travail
    d'images animées.

  4. Enfin, le format "svg" qui est utilié pour enregistrer des icênes, des logos ou des dessins.
    Contrairement aux trois formats précédents, qui sont pixélisés on peut modifier le format
    du "svg" ce qui permet de redimensionner une image sans effet de pixelisation.
    Il s'agit d'un format "vectoriel".

Il y a des images libre de droit sur de nombreux site dont comme par exemple le site Unsplash.

Exemples concrets d'insertion d'images sur une page HTML :

Pour insérer l'image il faut créer un dossier image au sein du dossier sur lequel on travail
ou utiliser la méthode du "glisser/déposer"

Ensuite, on peut coder, entre deux balises de paragraphes"p", on ajoute la balise qui va contenir l'image :
avec une valise "img/". Le slash à la fin de la balise est important car il s'agit d'une balise autofermante
ou balise orpheline. Elle se suffit à elle même et n'a pas besoin d'un balise fermante pour indiquer au navigateur
la fin de l'élèment sur la page.


Néanmoins pour que ça fonctionne correctement il faut ajouter quelques attributs
Dans un premier temps il faut indiquer la source de l'image -l'endroit où elle est stockée-
pour cela on utilise l'attribut "src = dossierImage/imagechoisie.jpg". En fait c'est
un fonctionnement simmilaire à celui des hyperliens : on indique le chemin pour trouver l'image ou
l'adresse URL si celle ci se trouve sur autre site web.


Dans un second temps, il est utile d'ajouter l'attribut alt pour le texte alternatif.
Il est nécéssaire de le renseigner car :

  • Aider à la navigation :cette description permet aux personnes malvoyantes decomprendre à quoi l'image fait réfèrence.

  • Remplacer une image non chargée :le texte alternatif sera affiché si l'image ne peut pas être téléchargée correctement
    par le navigateur.

  • Aider au référencement : ,lorsque le texte alternatif est bien rédigé, il sera untilisé par les moteurs de recherche
    pour le référencement.
